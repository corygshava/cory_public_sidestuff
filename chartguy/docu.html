Create beautiful bar, line, and scatter charts with simple configs.

1. Bar Chart

The bar chart visualizes data with rectangular bars.


chart.create('bar', {
  labels: ['Jan', 'Feb', 'Mar'],
  values: [10, 20, 15],
  color: '#2c7be5'
});
[Bar chart preview]

2. Line Chart

The line chart connects data points with a continuous line.


chart.create('line', {
  labels: ['Q1', 'Q2', 'Q3'],
  values: [5, 15, 25],
  color: '#47c363'
});
[Line chart preview]
